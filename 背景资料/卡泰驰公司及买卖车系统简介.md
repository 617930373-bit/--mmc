# 卡泰驰公司及买卖车系统简介

**卡泰驰公司简介**

我现在在卡泰驰（CARTech）——海尔集团旗下汽车产业互联网平台，定位中高端二手车直营连锁与生态赋能，以“车况、价格、服务三透明”为核心，通过**直营\+加盟合伙人模式**，搭建全国标准化二手车交易服务网络，为车商提供数字化、标准化、生态化全链路赋能。

**卡泰驰买卖车系统（我主要负责的系统整体功能模块简介）**

我现在在负责「买卖车系统（车商端/MMC）」——MMC采销服数字化管理平台，是面向\*\*直营门店与加盟车商的一站式门店经营管理SaaS，核心管“人、车、客、单、钱”：

主要功能模块包括：

1. **门店与员工管理**：多门店组织架构、岗位权限、人单合一KPI、考勤绩效、合伙人分润。

2. **车辆全生命周期管理**：收车评估、412项检测、整备建档、定价上架、库存调度、跨店调车、成本毛利核算。

3. **客户与销售管理（CRM）**：线索分配、客户画像、跟进记录、成交转化、售后维系。

4. **销单与交易闭环**：线上线下订单统一、定金/尾款/合同电子化、过户代办、金融保险对接、发票与结算。

5. **数据驾驶舱与运营**：实时营收、库存周转、毛利分析、车源热度、员工效能、智能定价辅助。

6. **生态协同**：打通卡泰驰商城C端流量、全国车源共享、集采供应链、金融/保险/售后赋能。

**我负责卡泰驰买卖车系统中的客户管理模块，以下是功能介绍**

**客户管理功能模块核心要解决线索到客户、**

7. 模块定位：面向卡泰驰直营 \& 加盟二手车门店的销售、销售经理、店长，聚焦二手车整车销售场景，实现线索到成交全链路客户数字化管理，规范销售跟进动作，提升门店转化效率。

8. 核心流程：线索多渠道录入→自动 / 手动，自动/手动分配给销售→销售持续跟进（可合理按照用户意向制定下次跟进计划）→到店→成交建档 / 弃单归档，支持客户公海流转，个人战败客户后，客户释放至本店下公海池，供其他销售领取跟进；

9. 核心功能：

- 线索与客户档案：线索导入、客户全信息建档、意向等级标签、客户意向等级管理、公私海管理、分配客户、转移客户归属、老客户回访；

- 销售跟进：跟进记录、待办提醒；

- 转化管理：成交关联销单

- 权限：数据隔离、离职客户交接（规划但暂未实现）

- 数据报表：线索、跟进、转化率、员工业绩看板

- 特色：深度联动门店车辆库存、销售订单模块，适配二手车门店多店协同模式

- 模块边界：仅对内车商员工使用，不面向 C 端车主；不含财务、车辆检测功能
